# 🤖 AI-Driven Article Search & WhatsApp Automation Bot

An advanced automation workflow built to streamline research and information delivery. This project leverages **n8n** and **AI-driven logic** to fetch, process, and deliver articles directly to WhatsApp.

## 🚀 Key Features
- **Automated Research:** Searches for relevant articles based on user-defined keywords.
- **Smart Integration:** Uses **UltraMsg API** for seamless WhatsApp delivery.
- **Local-to-Cloud Tunneling:** Deployed via **ngrok** to maintain a persistent online presence from a local environment.
- **Data Logging:** Automatically logs all search activities and results into **Google Sheets** via HTTP nodes.

## 🛠️ Tech Stack
- **Automation Engine:** n8n (Node.js based)
- **Networking:** ngrok (Tunneling)
- **Languages:** JavaScript / Node.js
- **API Integration:** UltraMsg, Google Sheets API, Webhooks
- **Backend:** Local Hosting on Dell Latitude 5490

## 📂 Project Structure
- `index.html`: Web interface for monitoring.
- `workflow.json`: Complete n8n workflow export for easy import.
- `README.md`: Project documentation and setup guide.
